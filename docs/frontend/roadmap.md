# Frontend Roadmap

## Phase 1: Frontend Foundation & Marketing ✅
- React 19 / Vite / Tailwind v4 Setup
- Global Design System & Typography
- Motion System & Animation Primitives
- Premium Landing Page
- Interactive High-Fidelity Simulators

## Phase 2: Authentication (Immediate Next)
- JWT interceptor setup (Axios).
- `/login` and `/register` views.
- `AuthProvider` and `ProtectedRoute` components.

## Phase 3: Dashboard Shell
- Authenticated application shell layout (`/app`).
- Sidebar navigation (collapsible).
- Header with user profile and quick actions.
- Breadcrumb navigation system.

## Phase 4: Core ERP Modules (Iterative)
1. **Products Module**: Catalog, variants, SKUs, pricing.
2. **Inventory Module**: Stock levels, warehouses, stock movements.
3. **Customers Module**: Basic CRM, interaction history.
4. **Orders (Sales) Module**: Order-to-cash workflow.
5. **Production Module**: Manufacturing runs, BOMs (Bill of Materials).
6. **Reports Module**: Recharts integration, dashboard metrics.

## Phase 5: Polish & Deployment
- Settings Module (User preferences, company config).
- E2E Testing (Playwright / Cypress).
- Full Backend Integration Audit.
- Production Build & Deployment.
- **v1 Release**
